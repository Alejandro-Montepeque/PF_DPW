# 🧩 Proyecto Final - Soluciones de Desarrollo y Seguridad para E-commerce

Este es el proyecto final para la materia de [Diseño de Painas Web].  
Consiste en un sitio web informativo sobre la **venta de servicios de desarrollo web**, **diseño responsivo** y **ciberseguridad para plataformas e-commerce**.

## 🚀 Tecnologías utilizadas

- HTML5
- CSS3
- Bootstrap 5 (vía NPM)
- Font Awesome (íconos)
- NPM (gestión de dependencias)
- Live Server (para entorno de desarrollo local)

## 🎯 Objetivo del proyecto

Desarrollar un sitio web con diseño moderno y responsivo, enfocado en mostrar:

- Servicios de desarrollo de software para e-commerce
- Diseño web adaptable a dispositivos móviles
- Soluciones en ciberseguridad para tiendas en línea
